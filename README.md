# Touch-Typing
Touch-typing speed tested and improve the agility and accuracy of touch typing using advanced ai/ml algorithms.
backend.py -- initial python file that is supposed to run a flask extention to organise the overall website structure.
index.html -- supposed to be a frontend - a beautiful interface to the users.
style.css -- to create css styling assistance to the html content of the website
word_coll.py -- to collect and organize words from the provided link, remove repetitions.
words.py -- to store the words taken from word_coll.py .
